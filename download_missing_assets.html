Unduh Aset yang Diperlukan

Berikut adalah daftar file yang perlu diunduh untuk melengkapi sistem PPDB:

File JavaScript (Simpan di folder assets/js/):

  • bootstrap.bundle.min.js - File ini diperlukan untuk fungsi Bootstrap
  • jquery.min.js - File ini diperlukan untuk fungsi jQuery

File Gambar (Simpan di folder assets/img/):

  • logo.png - Logo sekolah Anda

    • Catatan: Untuk logo.png, Anda perlu menyiapkan file logo sekolah Anda sendiri. Pastikan ukuran file tidak terlalu besar (disarankan di bawah 200KB) dan format PNG dengan latar belakang transparan.

Petunjuk Penyimpanan:

  1. Buat folder assets/js/ jika belum ada
  2. Buat folder assets/img/ jika belum ada
  3. Unduh file JavaScript (.js) dan simpan di folder assets/js/
  4. Simpan file logo sekolah Anda di folder assets/img/ dengan nama logo.png
  5. Pastikan nama file sesuai dengan yang disebutkan di atas

Penting:

  • Jangan mengubah nama file setelah diunduh
  • Pastikan semua file berada di folder yang benar
  • Jika ada masalah, periksa kembali struktur folder dan nama file
